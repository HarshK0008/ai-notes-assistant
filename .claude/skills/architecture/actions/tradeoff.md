# Tradeoff Action

1. Identify the decision and the real options on the table from $ARGUMENTS or the
   current conversation. If fewer than two genuine options exist, say so instead
   of inventing a false choice.
2. For each option, state:
   - **Pros**
   - **Cons**
   - **Cost to reverse later** (cheap, moderate, expensive)
3. Give a recommendation with a one-sentence justification — don't just list
   options and leave the decision unmade.
4. If the human accepts a recommendation and it meets the "When to Write an ADR"
   bar in @.claude/knowledge/ai-interaction.md, offer to run the `decide` action.
