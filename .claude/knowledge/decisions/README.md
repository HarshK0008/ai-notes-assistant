# Architecture Decisions

An append-only log of architecture decision records (ADRs) — one file per
decision, numbered sequentially (`0001-`, `0002-`, ...). Never edit history;
if a decision is reversed, add a new ADR that supersedes the old one and
update the old one's Status to point to it.

See `.claude/knowledge/ai-interaction.md` for guidance on when a decision
warrants an ADR. Use `/architecture decide` to draft one from
`.claude/templates/adr.template.md`.

`0001-example.md` shows the expected shape and can be deleted once a project
has real decisions recorded.
